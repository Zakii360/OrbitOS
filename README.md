# 🪐 OrbitOS

A browser-based OS built with pure HTML, CSS, and JavaScript. Crafted by **360 Digital, Co.**

## Features

- 🌐 **360 Browser** — iframes 360-search.com as the default browser
- 📁 **File Explorer** — virtual filesystem
- 💻 **Terminal** — working shell with `help`, `ls`, `neofetch`, `clear`, and more
- 📝 **Text Editor** — with line count and save
- 🎵 **Music Player** — animated vinyl, track controls
- ⚙️ **Settings** — wallpaper switcher, system info, toggles

## UI

- Draggable, resizable windows
- Minimize / maximize / close controls
- Bottom Windows-style taskbar
- Start menu with app grid
- Right-click context menu on desktop
- Toast notifications
- Live clock

## Stack

Pure vanilla HTML/CSS/JS — zero dependencies, zero frameworks.

## Usage

Just open `index.html` in any browser. No build step needed.
